## Functions

arange
```
np_array = np.arange(0,11,2)

print(np_array)
```
linspace - generate evenly distributed 100 samples between 0 and 5
default value is 50
```
np_array = np.linspace(0,5,100)

print(np_array)
```

zeros() - create a zero matric
```
np_array = np.zeros([2,3])

print(np_array)
```
ones() - same as above with 1s
